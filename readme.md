
# Scanova QR Generator

This is a QR Code Generator Made using Python Streamlit.

## 🚀  Features

 1. It takes the input URL.
 2. Converts it into a QR Code.
3. User can scan the QR and they will be redirect to the webpage
4. They can download the QR In the png, jpg, jpeg format

# 🌐 Live Demo

Web app live at: [https://scanova-qr-generator-15.streamlit.app/]



## Requirements

First download the libraries
```bash
  pip install streamlit
  pip install qrcode
```

To run this project run streamlit run app.py
